SM3 Alert RunBook – Verint - Agent Capture is not reachable

Alert Name: Agent Capture is not reachable

Alert Expression: round(increase(verint_alarms_agent_capture_unreachable_total{sev="WARNING"}[5m])) > 0

Category: Verint

Description: This alert occurs when Agent Capture systems are unreachable, as evidenced by increasing alarms in the last 5 minutes for a specific host.

Possible Cause(s):
