SM3 Alert RunBook – Verint - Archive is unable to access a configured SAN or fileshare path

Alert Name: Archive is unable to access a configured SAN or fileshare path

Alert Expression: round(increase(verint_alarms_archive_fileshare_connect_error_total{sev="WARNING"}[5m])) > 0

Category: Verint

Description: Triggers if there is an issue accessing SAN or fileshare paths, as shown by the rise in ArchiveFileshareConnectError alarms for the host over the last 5 minutes.

Possible Cause(s):

Network configuration issues
SAN or NAS accessibility problems
Permission or credential issues
